# MEAN Stack Application - Login & Data Management
# Project Overview
A MEAN stack application demonstrating user authentication, data management from external APIs, and inline editing capabilities. Built with Angular 21, this application showcases modern web development practices including standalone components, signals for state management, and responsive design.

# Key Features
Secure Authentication - Login system with route protection using hardcoded credentials (admin/password123)

Real Data Integration - Fetches and displays live data from external REST API (api.restful-api.dev/objects)

Inline Editing - Full CRUD operations with inline editing for all object properties (name, color, capacity, price)

Modern Angular - Utilizes Angular 21 with signals, standalone components, and modern reactive patterns

Responsive Design - Mobile-friendly interface with clean, professional styling

# Technical Stack
Frontend: Angular 21, TypeScript, Signals, Standalone Components

Styling: Custom CSS with Bootstrap-inspired design

API: External REST API integration with proper error handling

Architecture: Component-based with services for business logic

# Quick Setup
Clone repository and run npm install

Start development server with ng serve

Access at http://localhost:4200

Login using credentials: admin / password123

# Project Structure
The application follows Angular best practices with organized modules:

Auth services and guards for security

Login component with form validation

Report component with data table and inline editing

API service for external communication

Route protection ensuring authenticated access
